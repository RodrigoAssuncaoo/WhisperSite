What is my project about?

This website is part of a project called Whisper. The project revolves around a tourism company offering tours in Lisbon and its surroundings. It will include a MySQL database, an application developed in Android Studio, and a functional website where users can contact the company to book their tours around Lisbon, with various types of routes and guides for any destination they wish to explore.

Tools I used in my project

Visual Studio Code

Visual Studio Code was the tool that allowed me to program in multiple programming languages like MySQL, HTML, CSS, and JavaScript. Its features, such as the integrated terminal and support for extensions, made the development process much smoother and more efficient

Android Studio

Android Studio allowed me to use Java programming, which enabled me to create the splash screen for my project.

Xampp

XAMPP made it easy to set up a local server for my project, allowing me to run MySQL and test PHP scripts locally.
